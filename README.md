# Movingthebaytoken.com

Crypto wallet for moves - San Francisco's premier moving company that accepts crypto payments and rewards customers with BayMove Tokens (BMT).

## Overview

A crypto-powered moving service platform for the San Francisco Bay Area that integrates cryptocurrency payments with a custom loyalty token. Customers receive 5-10% of their move cost in BMT tokens, which can be redeemed for service discounts or traded on DEX platforms.

## Features

- **Crypto Payments** - Accept Solana (SOL) and Ethereum (ETH) via NOWPayments
- **BayMove Token (BMT)** - Earn loyalty tokens redeemable for discounts
- **Online Booking** - Schedule moves with date/time selection
- **Service Coverage** - San Francisco, Oakland, San Jose, and Marin County

### Token Rewards

| BMT Balance | Reward |
|-------------|--------|
| 100 BMT | $10 off packing services |
| 500 BMT | Free full-day labor upgrade |

## Tech Stack

- **Frontend**: Tailwind CSS (CDN)
- **Typography**: Inter (Google Fonts)
- **Wallet**: Phantom Wallet integration
- **Blockchain**: Solana
- **Payments**: NOWPayments

## Installation

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd movingthebaytoken
   ```

2. Run locally using any of these methods:

   **Python:**
   ```bash
   python3 -m http.server 8000
   ```

   **Node.js:**
   ```bash
   npx http-server
   ```

   Then visit: `http://localhost:8000`

No build step or dependencies required - this is a static HTML site.
